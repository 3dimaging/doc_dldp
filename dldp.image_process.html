Title: color normalization¶

Created: 10-31-2019 Python-Version: 3.5

Description:¶

This module is used to do the color normalization for image patches.

Input: raw image patches. Output: normalized image patches.

Request: this module relys on:

https://github.com/wanghao14/Stain_Normalization https://github.com/Peter554/StainTools
