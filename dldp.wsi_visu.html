dldp.wsi_visu.WSI_viewer module¶

Title: wsi visualization¶

Created: 10-31-2019 Python-Version: 3.5

Description:¶

This module is used to view the WSI, its mask and heatmap overlay.

Note:¶

The level of display resolution depends on the memory of the workstation used. The default level is 5 (32x downsampled from the level 0 WSI image).

class dldp.wsi_visu.WSI_viewer.WSI_viewer(WSI_path, Xml_path, Dimension_path, Mask_truth_path='', Heatmap_path='')¶

Bases: object

The wsi_viewer class is used to display the wsi image, ground truth, predicted heatmap and their overlays.

Parameters

  • WSI_path (str) – the path of a WSI image

  • Xml_path (str) – the path of a xml file, the ground truth (outline the tumor regions) of a WSI image

  • Dimension_path (str) – the path of npy file I generated to store the dimensions of WSI image and tissue region.

  • Mask_truth_path (str) – the path of generated binary mask files from xml files.

  • Heatmap_path – the path of predicted heatmap showing the scores of possible tumor region.

:type Heatmap_path

ivar: contours of tissue region, tumor region (from ground truth), and

heatmap on WSI image.

vartype: png

  • create an instance of the object

    for example, viewer = WSI_viewer()

  • display the combined contours of tissue region and tumor

    region viewer.combined_contour_on_wsi()

  • display heatmap over contours of tumor region

    viewer.display_heatmap()

  • generate binary mask flies:

    viewer.mask_generation()
